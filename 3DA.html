Cursed Noodle

Beginnend met het maken van de omgeving: achtergrond, HDRI in sky object en de glazen kom, knop etc. Begon ik met de noodle te maken. Ik begon eerst met wat een van de gegeven video's uitgelegd had (waarvan ik ook best wat heb geleerd over het fracture object en emitters.) maar zag dat de bollen die gecreëerd werden soms vast zatten in de lucht. Dus ben ik overgeschakelt naar een op voorand gemaakte noodle via een array object.

Daarna begon ik met de hand, deze heb ik van MakeHuman gehaald (geweldig programma voor menselijke lichamen.) Met de bone tool maakte ik het skelet van de hand, om dan het object te "skinnen" aan het skelet. Deze kon ik dan makkelijk animeren voor de knop in te drukken.

Blender

Links is RAW EXR bestand, rechts is aangepast via ACES in Resolve.

De blender heb ik gemaakt met standaard objecten en vector object in een revolve object geplaatst. Daarna heb ik de cubus geplaatst in een fracture object, en een paar rigidbody instellingen geanimeerd zodat de cubus een deftig afbrekend effect kreeg tijdens zijn simulatie in de blender. Renderen was gedaan in Prorender, opgeslagen als EXR om dan in Davinci resolve door het ACES colour pipeline gebracht te worden.
